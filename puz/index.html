HTML5 Across Lite Reader

Simply drag & drop a .puz file here, and I'll make you a crossword. No files will be sent to the server, this is entirely done using your modern Web browser. Note that this only currently works with the following HTML5-compatible browsers:

  • Google Chrome 4+
  • Mozilla Firefox 3.6+

Note 2021-09-04: The original web app used to live at Derek Slager's site, but at some point it stopped working. I (Leo Stein) grabbed a static archived version from the Internet Archive and am mirroring it here for personal convenience. All credit goes to Derek Slager for originally developing and hosting the page.

UP/DOWN/LEFT/RIGHT Moves around the grid or changes direction.
CTRL + UP/DOWN/LEFT/RIGHT Change current direction.
SHIFT + UP/DOWN/LEFT/RIGHT Moves to the previous or next clue.
HOME/END Moves to the first or last square in the current word.
SHIFT + HOME/END Moves to the first or last square in the puzzle.
TAB/ENTER Moves to the next word.
SHIFT + TAB/ENTER Moves to the previous word.
DELETE Erases the current square value.
BACKSPACE Erases the current square value and moves to the previous square, if possible.
SPACE Erases the current square and moves to the next square, if possible.
INSERT/F2 Opens the rebus entry dialog.
